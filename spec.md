# Specification

## Summary
**Goal:** Add Tinder-style profile card headers to each economist's StatementSlide and EconomistRevealCard components, displaying name, era/dates, portrait, and school-of-thought tag in Tinder's classic layout.

**Planned changes:**
- Add a Tinder-style profile card header to each economist entry in `StatementSlide` and `EconomistRevealCard`
- Display economist's name in large bold white typography overlaid at the bottom-left of the card header
- Show era/date range (e.g. "1723–1790") and school-of-thought tag as subtitle labels
- Use a full-bleed illustrated portrait image area at the top with a dark gradient overlay at the bottom
- Use the new stylized economist portrait images as the card header backgrounds
- Keep styling consistent with the existing dark/gold OKLCH design system

**User-visible outcome:** Each economist card now features a visually rich Tinder-style profile header with portrait, name, lifespan dates, and school of thought — making the app feel like a true dating-app parody for economists.
